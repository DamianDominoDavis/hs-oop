class Node

Elements of a Linked List are instances of the Node class with two instance variables:

  • data — the value an element contains and represents, never 
    None
  • next — a reference pointing to the next Node element in a Linked List, or 
    None

Import your Node class into your LinkedList class to use Nodes and their instance vairiables to build the functionality of the LinkedList class.

class LinkedList

Linked Lists are iterable collections of Nodes linked in one direction from beginning to end. It has a minimum of one instance variable:

  • head — the first Node, or 
    None

Each instance of LinkedList handles the manipulation of its own elements. Methods of the LinkedList class should include:

  • is_empty(self) — returns 
    True
    if this list has no elements, else returns 
    False
  • add_head(self, value) — creates a Node containing value, and sets it as the head of the list
  • add_tail(self, value) — creates a Node containing value, and places it as the end of the list
  • __iter__(self) — the iterator for this LinkedList
  • __str__(self) — the string representation of this LinkedList
  • size(self) — returns the integer number of elements in this LinkedList
  • remove_first(self, value) — iterates over elements until encountering a Node of data equal to value; unlinks the node (preserving the remaining links) and returns 
    True
    if found, else does not alter the list and returns 
    False
  • has_cycle(self) — returns 
    True
    if this is a circularly LinkedList, else returns 
    False
  • sort_asc(self) — sorts the elements of this list by ascending element value
